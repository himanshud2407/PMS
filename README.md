## 🚀 Run Locally

**Prerequisites:**  Node.js (v18+)

1. **Clone & Install**:
   ```bash
   npm install
   ```
2. **Environment Setup**:
   Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
3. **Launch**:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Frontend**: React, Tailwind CSS, Framer Motion
- **Icons**: Lucide React
- **Animations**: GSAP, Motion/React
- **Type**: TypeScript
